# Class Projects Collection

Complete project structure for learning and development.

## Structure

- **frontend/** - Frontend applications
- **backend/** - Backend services  
- **login-service/** - Authentication service
- **utilities/** - Utility applications
  - file-converter/
  - calculator/
  - humanizer/
  - chatbot/
- **class-projects/** - Academic projects

## Setup

1. Clone repository
2. Navigate to desired project folder
3. Follow individual README instructions

## Branches

Each component has its own branch for isolated development.
